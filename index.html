Pydroide

"Aprenda Python 3, com o mais poderoso IDE & interpretador Python 3 no Google Play"

Classificação na Playstore: 4.3 estrelas

O Pydroid 3 é o IDE educacional Python 3 mais fácil de usar e poderoso para Android.

Aviso:

"Em caso de problemas de tensorflow, instale "keras==2.6" em pip. Este problema foi introduzido no keras após o lançamento do Pydroid 3 e estamos trabalhando em uma correção."

Recursos

  • Intérprete Python 3 offline: não é necessária Internet para executar programas Python.
  • Gerenciador de pacotes Pip e um repositório personalizado para pacotes wheel pré-construídos para bibliotecas científicas aprimoradas, como numpy, scipy, matplotlib, scikit-learn e jupyter.
  • OpenCV já está disponível (em dispositivos com suporte a API Camera2).
  • TensorFlow e PyTorch também estão disponíveis.
  • Exemplos disponíveis prontos para uso para um aprendizado mais rápido.
  • Suporte completo ao Tkinter para GUI.
  • Emulador de terminal completo, com suporte para readline (disponível em pip).
  • Compilador C, C++ e até Fortran integrado projetado especialmente para Pydroid 3. Ele permite que o Pydroid 3 construa qualquer biblioteca a partir do pip, mesmo que esteja usando código nativo. Você também pode construir e instalar dependências a partir de uma linha de comando.
  • Suporte Cython.
  • Depurador PDB com pontos de interrupção e relógios.
  • Biblioteca gráfica Kivy com um novo backend SDL2 brilhante.
  • Suporte PyQt5 disponível no repositório Quick Install juntamente com suporte matplotlib PyQt5 sem necessidade de código extra.
  • Suporte ao Matplotlib Kivy disponível no repositório Quick Install.
  • Suporte pygame 2.

Recursos do editor:

  • Previsão de código, recuo automático e análise de código em tempo real, como em qualquer IDE real.
  • Barra de teclado estendida com todos os símbolos que você precisa para programar em Python.
  • Destaque de sintaxe e temas.
  • Abas.
  • Navegação de código aprimorada com gotos interativos de atribuição/definição.
  • Compartilhamento de um clique no Pastebin.
    • Os recursos riscados estão disponiveis apenas na versão premiun

Considerações básicas
O Pyrdroid 3 requer pelo menos 250 MB de memória interna livre. 300 MB + é recomendado. Mais se você estiver usando bibliotecas pesadas, como scipy. Para executar a depuração, coloque o(s) ponto(s) de interrupção clicando no número da linha. Kivy é detectado com "importar kivy", "de kivy" ou "#Pydroid execute kivy". PyQt5 é detectado com “import PyQt5”, “from PyQt5” ou “#Pydroid run qt”. O mesmo para sdl2, tkinter e pygame.
Existe um modo especial "#Pydroid run terminal" para garantir que seu programa seja executado no modo terminal (isso é útil com matplotlib que é executado automaticamente no modo GUI)

Por que algumas bibliotecas no pydroide são somente premium?
"Essas bibliotecas eram extremamente difíceis de portar, então tivemos que pedir a outro desenvolvedor para fazer isso. Sob acordo, seus forks dessas bibliotecas são fornecidos apenas para usuários premium."
Se você tem conhecimentos sobre a criação de forks e gostaria de desenvolver de forma gratuita para esta aplicação entre em contacto com os desenvolvedores
Informação Legal
Alguns binários no Pydroid 3 APK são licenciados sob (L)GPL, envie-nos um e-mail para obter o código-fonte.
As bibliotecas Python puras GPL empacotadas com o Pydroid 3 são consideradas já na forma de código-fonte.
O Pydroid 3 não inclui nenhum módulo nativo licenciado pela GPL para evitar a importação automática deles. O famoso exemplo dessa biblioteca é o GNU readline, que pode ser instalado usando o pip.
As amostras disponíveis no aplicativo são gratuitas para uso educacional com uma exceção: elas, ou seus trabalhos derivados, não podem ser usadas em nenhum produto concorrente (de forma alguma). Se você não tiver certeza se seu aplicativo é afetado por essa restrição, sempre peça uma permissão por e-mail.

Qpython

O QPython 3x é principalmente para usuários experientes em Python.

Classificação na Playstore: 3.8 estrelas

QPython é o mecanismo Python para android. Ele contém alguns recursos surpreendentes:

  • Interpretador python
  • Ambiente de tempo de execução
  • Editor
  • Biblioteca QPYI e SL4A
Isso facilita o uso do Python no Android. E é Grátis.

O QPython já possui milhões de usuários em todo o mundo e também é um projeto de código aberto.
Para diferentes cenários de uso, o QPython possui duas ramificações, a saber, Qpython Ox e 3x.
QPython Ox é voltado principalmente para alunos de programação e fornece recursos mais amigáveis ​​para iniciantes.
 Este é o QPython 3L, é principalmente para usuários experientes de Python e fornece alguns recursos técnicos avançados.

Recursos:

  • Intérprete offline do Python 3: não é necessária Internet para executar programas Python
  • Ele suporta a execução de vários tipos de projetos, incluindo: console, SL4A, webapp
  • Leitor de código QR conveniente para transferir códigos para o seu telefone
  • QPYPI e um repositório personalizado para pacotes de roda pré-construídos para bibliotecas científicas aprimoradas, como numpy, scipy, matplotlib, scikit-learn etc.
  • Editor fácil de usar
  • CAMADA DE SCRIPT INTEGRADA E ESTENDIDA PARA A BIBLIOTECA ANDROID (SL4A): permite que você direcione o trabalho androide com o Python
  • Boa documentação e suporte ao cliente

Recursos SL4A
Com os recursos do SL4A, você pode usar a programação Python para controlar o trabalho do Android:

  • API de aplicativos Android, como: Aplicativo, Atividade, Intenção e início de atividade, SendBroadcast, PackageVersion, Sistema, Toast, Notify, Configurações, Preferências, GUI
  • Gerenciador de recursos do Android, como: contato, localização, telefone, SMS, ToneGenerator, WakeLock, WifiLock, área de transferência, NetworkStatus, MediaPlayer
  • Terceiras integrações de aplicativos, como: código de barras, navegador, SpeechRecongition, SendEmail, TextToSpeech
  • Hardwared Manager: Carmer, Sensor, Campainha e Volume de Mídia, Brilho da Tela, Bateria, Bluetooth, Força do Sinal, WebCam, Vibrar, NFC, USB
